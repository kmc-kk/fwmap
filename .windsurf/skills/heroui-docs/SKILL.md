When working on UI in this repository, use HeroUI v3 as the primary UI library.
Reference these docs when needed:
- https://v3.heroui.com/react/llms-full.txt
- https://v3.heroui.com/react/llms-components.txt

Prefer HeroUI components, patterns, and theming conventions over introducing other UI libraries.
